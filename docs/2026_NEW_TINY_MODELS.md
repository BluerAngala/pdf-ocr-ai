## 🚀 快速开始

### 1. 安装依赖
```bash
pip install llama-cpp-python --no-cache-dir
```

### 2. 下载模型（以Gemma 4 E2B为例）
```bash
# 方式1：使用脚本（推荐）
python scripts/download_model.py --model gemma-4-e2b

# 方式2：手动下载
# 访问 https://huggingface.co/google/gemma-4-e2b-it-GGUF
# 下载 gemma-4-e2b-it-Q4_K_M.gguf
# 放到 models/ 目录
```

### 3. 使用模型
```python
from src.local_llm import LocalLLM, LLMConfig

# 配置
config = LLMConfig(
    model_path="models/gemma-4-e2b-it-Q4_K_M.gguf",
    context_length=4096,  # 减小以节省内存
    max_tokens=512
)

# 加载模型
llm = LocalLLM(config)
llm.load_model()

# 检查OCR文本
ocr_text = "你的OCR识别结果..."
result = llm.analyze_ocr_text(ocr_text, task="check")
print(result)
```
