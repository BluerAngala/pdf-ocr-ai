# 公积金业务辅助工具软件开发需求

## 一、项目概述

本项目旨在开发一套公积金业务辅助工具，主要服务于非诉审查组和强制执行组两个业务团队，通过自动化技术（OCR识别、RPA等）提升案件处理效率，减少人工操作错误。

---

## 二、功能模块详细需求

### 模块1：非诉阶段 - PDF文件批量重命名工具 ✅ 已完成（持续优化中）

#### 1.1 背景说明

使用"模拟机械手"RPA协助进行网立，网立过程中RPA根据台账的指令选择对应的文件材料进行上传。因此每个文件材料须有准确的命名与台账相匹配，才能被RPA精准识别。由于案件量大，文件材料多，需要特定的插件/小程序批量对扫描后的文件材料进行重命名。

#### 1.2 功能范围

支持多种文件类型的批量重命名，每种类型对应不同的输入目录和命名规则：

| 类型 | 上级目录 | 命名规则 | 识别内容 |
|------|---------|---------|---------|
| 责催文件 | `Z:\LNLFDOC\法院RPA\输入文件\责催` | `{序号}-责催-{责令号}` | PDF首页责令号 |
| 非诉授权书 | `Z:\LNLFDOC\法院RPA\输入文件\非诉授权书` | `{公司名称}` | PDF中的公司名称 |
| 所函 | `Z:\LNLFDOC\法院RPA\输入文件\所函` | `{公司名称}` | PDF中的公司名称 |
| 台账 | `C:\Users\Administrator\Desktop\法院RPA` | `台账.xlsx` | - |

**命名示例：**

| 原文件名 | 正确命名后 |
|---------|-----------|
| 1-责催-穗公积金中心越秀责字（2025）916号.pdf | 1-责催-穗公积金中心越秀责字（2025）916号.pdf |
| 授权书.pdf | 广州市三英温泉酒店投资有限公司.pdf |
| 所函.pdf | 广州市三英温泉酒店投资有限公司.pdf |

#### 1.3 责催文件重命名规则

**输入：**
- 扫描的责催PDF文件（存放于 `Z:\LNLFDOC\法院RPA\输入文件\责催`）
- 台账Excel文件（存放于 `C:\Users\Administrator\Desktop\法院RPA\台账.xlsx`）

**处理逻辑：**
1. 读取台账Excel，获取序号与责令号的对应关系
2. OCR识别PDF首页的责令书案号
3. 文本规范化处理（统一括号、纠错等）
4. 匹配台账中的责令号
5. 按规则重命名PDF文件

**命名格式：**
```
{序号}-责催-{责令号}
```

**示例：**
```
1-责催-穗公积金中心越秀责字（2025）916号
```

#### 1.4 授权委托书重命名规则

**输入目录：** `Z:\LNLFDOC\法院RPA\输入文件\非诉授权书`

**处理逻辑：**
1. OCR识别PDF全文内容
2. 智能提取公司名称（通过关键词定位，如"委托单位"、"委托人"等）
3. 以识别到的公司名称进行命名

**命名格式：**
```
{公司名称}
```

**示例：**
```
广州市三英温泉酒店投资有限公司.pdf
```

#### 1.5 所函重命名规则

**输入目录：** `Z:\LNLFDOC\法院RPA\输入文件\所函`

**处理逻辑：**
1. OCR识别PDF全文内容
2. 智能提取公司名称（通过关键词定位，如"致"、"关于"等）
3. 以识别到的公司名称进行命名

**命名格式：**
```
{公司名称}
```

**示例：**
```
广州市三英温泉酒店投资有限公司.pdf
```

#### 1.6 OCR预处理逻辑（文本规范化）

为提高识别准确率，需进行以下预处理：

1. **统一括号格式**
   - 默认统一使用中文括号（`（）`）
   - 支持用户在设置中手动切换为英文括号（`()`）
   - 匹配时自动统一括号格式后再进行比较

2. **忽略空格和换行**
   - 去除识别文本中的多余空格和换行符

3. **修正识别错误文字**
   - 预设常见OCR错误词库（如"责行"→"责令"、"公积全"→"公积金"等）
   - 支持用户在设置中自定义添加纠错映射
   - 纠错规则按优先级应用

4. **案号格式规范化**
   - 识别案号后，统一转换为中文括号格式

5. **Excel匹配逻辑统一**
   - 匹配Excel表格时，同样统一括号格式，确保匹配成功

### 模块2：强制执行阶段 - 裁定信息识别与归纳工具

#### 2.1 功能目标

用于统计职工信息，识别裁定书中的关键信息，并与非诉组移送的表格进行匹配，填充OCR识别到的信息，导出完整表格。

#### 2.2 输出表格完整表头

| 字段名 | 数据来源 | 说明 |
|--------|---------|------|
| 区号 | 非诉组Excel | 原表数据 |
| 行政审查案号 | **OCR识别** | 从裁定书提取 |
| 责令号 | 非诉组Excel | 用于匹配的关键字段 |
| 被执行人 | 非诉组Excel | 原表数据 |
| 职工姓名 | 非诉组Excel | 原表数据 |
| 金额 | 非诉组Excel | 原表数据 |
| 审批员/法官助理 | **OCR识别** | 从裁定书提取 |
| 执行时间 | **OCR识别** | 从裁定书提取（准予执行日期） |

#### 2.3 需要OCR识别的信息字段

**从裁定书中提取（仅3个字段）：**
- 行政审查案号（行审案号）
- 审批员/法官助理
- 执行时间（准予执行日期）

#### 2.4 裁定书类型

**Verse1：** 一份裁定书仅包含一名职工

**Verse2：** 一份裁定书包含多名职工

**特殊情况：**
- 存在撤回执行的情况，同样需要识别和处理

#### 2.5 数据处理流程

1. **读取非诉组Excel**
   - 加载原始表格数据（区号、责令号、被执行人、职工姓名、金额）

2. **OCR识别裁定书**
   - 识别行政审查案号、审批员/法官助理、执行时间
   - 识别责令号（用于匹配）

3. **数据匹配**
   - **匹配字段**：OCR识别的责令号 ↔ Excel中的责令号
   - **匹配规则**：责令号匹配成功后，将OCR识别的3个字段填充到对应行
   - 支持模糊匹配（相似度阈值可调）

4. **信息导出**
   - 导出包含原表数据 + OCR识别数据的完整表格

#### 2.6 匹配规则详解

**匹配逻辑：**
```
OCR识别结果：
- 责令号：穗公积金中心越秀责字（2025）916号
- 行政审查案号：（2025）粤7101行审1234号
- 审批员/法官助理：张三
- 执行时间：2025年1月15日

↓ 匹配（按责令号）

非诉组Excel中责令号为"穗公积金中心越秀责字（2025）916号"的行

↓ 填充

该行填充：
- 行政审查案号 = （2025）粤7101行审1234号
- 审批员/法官助理 = 张三
- 执行时间 = 2025年1月15日
```

#### 2.7 现有问题及解决方案要求

| 问题描述 | 解决方案要求 |
|---------|-------------|
| Verse2（一份裁定书多名职工）中可能存在漏识别职工的情况 | 需优化OCR识别逻辑，确保多职工场景下所有职工信息都被识别 |
| 识别裁定信息时存在匹配不上非诉组给的表格导致遗漏职工的情况 | 需建立容错匹配机制，支持模糊匹配和人工校验功能 |

#### 2.8 模糊匹配配置

- **默认相似度阈值**：0.85（可配置）
- **支持预设阈值**：高(0.9) / 中(0.85) / 低(0.8)
- **支持手动调整**：用户可在设置中自定义阈值(0.7-1.0)

### 模块3：强制执行阶段 - 自动打印功能

#### 3.1 功能目标

根据Excel表格数据，自动搜索本地文件夹中的PDF文件并进行批量打印。

#### 3.2 打印配置项

| 配置项 | 说明 | 示例 |
|--------|------|------|
| Excel行范围 | 选择打印的数据行范围 | 从第2行到第100行 |
| 搜索列名 | 指定用于搜索PDF文件的Excel列 | "责令号" 或 "行政审查案号" |
| 搜索文件夹 | 指定存放PDF文件的本地文件夹路径 | `C:\案件PDF\` |
| 打印机选择 | 选择可用的打印机设备 | 打印机列表下拉选择 |
| 打印份数 | 每份文件打印的份数 | 1-99份 |
| 单双面 | 单面打印或双面打印 | 单面/双面 |

#### 3.3 打印流程

1. **加载Excel数据**
   - 用户选择Excel文件
   - 设置行范围（如：从第2行到第100行）
   - 选择搜索列名（如："责令号"）

2. **搜索PDF文件**
   - 根据选定列的值，在指定文件夹中搜索匹配的PDF文件
   - 匹配规则：文件名包含列值（支持模糊匹配）

3. **打印设置**
   - 选择打印机
   - 设置打印份数
   - 设置单双面

4. **执行打印**
   - 批量发送打印任务
   - 显示打印进度
   - 记录打印结果

#### 3.4 打印示例

```
Excel数据：
行号 | 责令号                          | 行政审查案号
-----|----------------------------------|---------------------------
2    | 穗公积金中心越秀责字（2025）916号 | （2025）粤7101行审1234号
3    | 穗公积金中心天河责字（2025）917号 | （2025）粤7101行审1235号

配置：
- 行范围：2-3行
- 搜索列：责令号
- 搜索文件夹：C:\案件PDF\
- 打印份数：2份
- 单双面：双面

执行：
1. 搜索包含"穗公积金中心越秀责字（2025）916号"的PDF文件
2. 搜索包含"穗公积金中心天河责字（2025）917号"的PDF文件
3. 找到后，各打印2份双面
```

#### 3.5 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 找不到对应PDF文件 | 跳过该行，记录到异常列表 |
| 打印机不可用 | 提示用户检查打印机 |
| 打印任务失败 | 重试3次，仍失败则记录异常 |
